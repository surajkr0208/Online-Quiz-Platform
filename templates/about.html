About QuizMaster

A Python-powered online quiz platform built to make learning interactive, fun, and rewarding. Built with Flask, designed with care.

🐍 Python 3
⚡ Flask
🎨 Vanilla CSS
💾 Sessions
🔌 REST API
📱 Responsive

🏗️

Built with Python & Flask

The backend is powered by Python 3 and the Flask micro-framework. Routes, session management, and AJAX endpoints are all handled server-side with clean, readable code.

🎯

Multiple Quiz Categories

Choose from Python Basics, Data Science, Web Development, Algorithms, Machine Learning, and General Knowledge — with more categories planned.

🌓

Dark & Light Mode

A fully implemented dark/light theme toggle that persists your preference using localStorage — no flash on reload, smooth transitions throughout.

⚡

AJAX-Powered Quizzing

Answers are submitted via JavaScript fetch API without page reloads, giving an instant, app-like feel with real-time feedback and explanations.

🔒

Session-Based State

Quiz progress, scores, and answers are tracked using Flask's server-side sessions, allowing secure, per-user state without a database.

📊

Score & Analytics

At the end of each quiz, get a detailed breakdown with your score percentage, grade, per-question review, and explanations for every answer.

🏛️ Architecture Overview

Backend

  • Flask routes for all pages
  • JSON-based quiz data store
  • Session for score tracking
  • REST endpoint for answers

Frontend

  • Jinja2 HTML templates
  • Vanilla CSS design system
  • AJAX with Fetch API
  • localStorage theme toggle
