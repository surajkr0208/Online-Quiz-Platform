# 🧠 QuizMaster - Online Quiz Platform

Professional Python-based quiz platform built with Flask.

## 🚀 How to Run

1. **Open Folder**: Open this directory in VS Code.
2. **Terminal**: Open a new terminal in VS Code.
3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
4. **Run App**:
   ```bash
   python app.py
   ```
5. **View Site**: Open [http://127.0.0.1:5000](http://127.0.0.1:5000)

## 🔐 Admin Access
- **URL**: [http://127.0.0.1:5000/admin](http://127.0.0.1:5000/admin)
- **Password**: `admin123`

## ✨ Features
- **150 Questions**: 25 questions in each of the 6 categories.
- **Admin Panel**: Create, Edit, and Delete quizzes and questions.
- **Dark/Light Mode**: Toggle available in the navbar.
- **Persistent Data**: Changes made in Admin are saved to `data/quizzes.json`.
